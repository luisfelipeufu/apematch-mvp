# 🏠 ApêMatch - MVP (Produto Mínimo Viável)

Projeto desenvolvido para a disciplina de Engenharia de Software II. O ApêMatch é uma plataforma que conecta usuários para moradia compartilhada baseando-se em compatibilidade de hábitos de convivência.

## 🚀 Funcionalidades do MVP
- **Autenticação:** Cadastro e Login blindados com criptografia de senha (`password_hash`) e validação de CPF.
- **Onboarding Inteligente:** Redirecionamento automático para novos usuários preencherem seus hábitos.
- **Algoritmo de Match:** Cálculo matemático de afinidade baseado em Fumo, Pets, Sono, Som, Visitas e Organização.
- **Interface Completa:** Front-end moderno com Navbar, visualização de Meu Perfil, Mockup de Mensagens (Chat) e vitrine de Imóveis Anunciados.

## 🛠️ Tecnologias Utilizadas
- **Front-end:** HTML5, CSS3 (Google Fonts: Poppins).
- **Back-end:** PHP 8.
- **Banco de Dados:** MySQL.
- **Hospedagem:** InfinityFree.
